# راهنمای ارائه‌دهندگان: اتصال مدل‌های هوش مصنوعی به RouteChi

> **خلاصه**: یک ارائه‌دهنده، یک اتصال به یک سرویس هوش مصنوعی است (مثل OpenAI، Anthropic، Google). برای استفاده از RouteChi به حداقل یک ارائه‌دهنده نیاز دارید.

---

## ارائه‌دهنده چیست؟

ارائه‌دهنده را مثل یک **شرکت مخابراتی** تصور کنید. همان‌طور که برای تماس گرفتن به یک شرکت مخابراتی نیاز دارید، برای استفاده از مدل‌های هوش مصنوعی به یک ارائه‌دهنده هوش مصنوعی نیاز دارید. RouteChi مثل تلفنی است که با **همه شرکت‌های مخابراتی** کار می‌کند — می‌توانید به‌صورت خودکار بین آن‌ها سوییچ کنید.

### انواع ارائه‌دهندگان

| نوع            | چیست                       | مثال‌ها                           | هزینه                 |
| -------------- | -------------------------- | --------------------------------- | --------------------- |
| **رایگان**     | بدون نیاز به پرداخت        | Kiro، OpenCode Free، Pollinations | ۰$                    |
| **کلید API**   | به یک کلید API نیاز دارید  | OpenAI، Anthropic، Google         | پرداخت به ازای استفاده |
| **OAuth**      | ورود با حساب شما           | Claude Code، GitHub Copilot       | اشتراک                |
| **وب کوکی**    | از نشست مرورگر شما استفاده می‌کند | ChatGPT Web، Gemini Web           | ۰$ (از حساب شما استفاده می‌کند) |

## شروع سریع: اتصال اولین ارائه‌دهنده

### گزینه A: ارائه‌دهنده رایگان (بدون کارت اعتباری)

1. داشبورد را در `http://localhost:20128` باز کنید
2. به **Providers** → **Add Provider** بروید
3. یکی از این ارائه‌دهندگان رایگان را انتخاب کنید:
   - **Kiro AI** — مدل‌های رایگان کلود (بدون نیاز به احراز هویت)
   - **OpenCode Free** — مدل‌های رایگان GPT (بدون نیاز به احراز هویت)
   - **Pollinations** — GPT-5، Claude، Gemini رایگان (بدون نیاز به کلید)
   - **LongCat** — ۱۰M توکن رایگان (اعتبار یک‌باره، نیاز به حساب + KYC)
   - **Cloudflare AI** — +۵۰ مدل، ۱۰K نورون/روز
4. روی **Connect** کلیک کنید
5. تمام! حالا دسترسی رایگان به هوش مصنوعی دارید.

### گزینه B: ارائه‌دهنده کلید API (پولی)

1. یک کلید API از وب‌سایت ارائه‌دهنده بگیرید:
   - **OpenAI**: https://platform.openai.com/api-keys
   - **Anthropic**: https://console.anthropic.com/
   - **Google**: https://aistudio.google.com/apikey
   - **DeepSeek**: https://platform.deepseek.com/
   - **Groq**: https://console.groq.com/
2. داشبورد را در `http://localhost:20128` باز کنید
3. به **Providers** → **Add Provider** بروید
4. ارائه‌دهنده خود را انتخاب کنید
5. کلید API خود را paste کنید
6. روی **Connect** کلیک کنید
7. تمام! حالا به مدل‌های آن ارائه‌دهنده دسترسی دارید.

### گزینه C: ارائه‌دهنده OAuth (اشتراک)

1. داشبورد را در `http://localhost:20128` باز کنید
2. به **Providers** → **Add Provider** بروید
3. ارائه‌دهنده خود را انتخاب کنید (مثلاً Claude Code، GitHub Copilot)
4. روی **Connect with OAuth** کلیک کنید
5. با حساب خود وارد شوید
6. تمام! حالا به مدل‌های اشتراک خود دسترسی دارید.

## نحوه اتصال یک ارائه‌دهنده (گام به گام)

### گام ۱: باز کردن داشبورد

در مرورگر خود به `http://localhost:20128` بروید.

### گام ۲: رفتن به Providers

روی **Providers** در نوار کناری کلیک کنید.

### گام ۳: کلیک روی Add Provider

روی دکمه **+ Add Provider** کلیک کنید.

### گام ۴: انتخاب ارائه‌دهنده

در لیست مرور کنید یا ارائه‌دهنده خود را جستجو کنید. روی آن کلیک کنید.

### گام ۵: وارد کردن اعتبارنامه‌ها

- **ارائه‌دهندگان رایگان**: بدون نیاز به اعتبارنامه — فقط روی **Connect** کلیک کنید
- **ارائه‌دهندگان کلید API**: کلید API خود را paste کنید
- **ارائه‌دهندگان OAuth**: روی **Connect with OAuth** کلیک کنید و وارد شوید

### گام ۶: تست اتصال

روی **Test Connection** کلیک کنید تا از کارکرد آن مطمئن شوید.

### گام ۷: تمام!

ارائه‌دهنده شما اکنون متصل است. می‌توانید از آن با `model: "auto"` استفاده کنید یا ارائه‌دهنده را مستقیماً مشخص کنید.

## استفاده از چندین ارائه‌دهنده

RouteChi با **چندین ارائه‌دهنده** بهترین کارکرد را دارد. این به شما می‌دهد:

- **Fallback خودکار** — اگر یک ارائه‌دهنده شکست بخورد، RouteChi گزینه بعدی را امتحان می‌کند
- **بهینه‌سازی هزینه** — RouteChi برای هر درخواست ارزان‌ترین ارائه‌دهنده را انتخاب می‌کند
- **بهینه‌سازی سرعت** — RouteChi برای هر درخواست سریع‌ترین ارائه‌دهنده را انتخاب می‌کند
- **بهینه‌سازی کیفیت** — RouteChi برای هر وظیفه بهترین ارائه‌دهنده را انتخاب می‌کند

### راه‌اندازی پیشنهادی

برای بهترین تجربه حداقل **۳ ارائه‌دهنده** متصل کنید:

1. **یک ارائه‌دهنده رایگان** (Kiro، OpenCode Free یا Pollinations) — همیشه در دسترس
2. **یک ارائه‌دهنده سریع** (Groq، Cerebras) — برای پاسخ‌های سریع
3. **یک ارائه‌دهنده باکیفیت** (OpenAI، Anthropic، Google) — برای وظایف پیچیده

سپس از `model: "auto"` استفاده کنید و RouteChi به‌صورت خودکار برای هر درخواست بهترین را انتخاب می‌کند.

## سؤالات متداول

### «آیا برای استفاده از RouteChi باید پرداخت کنم؟»

**خیر!** RouteChi رایگان و متن‌باز است. می‌توانید از ارائه‌دهندگان رایگان (Kiro، OpenCode Free، Pollinations) بدون پرداخت چیزی استفاده کنید. فقط اگر انتخاب کنید از ارائه‌دهندگان پولی استفاده کنید، هزینه می‌پردازید.

### «با کدام ارائه‌دهنده شروع کنم؟»

با **Kiro AI** شروع کنید — رایگان است، نیازی به کلید API ندارد و به شما دسترسی به مدل‌های کلود می‌دهد. سپس در صورت نیاز ارائه‌دهندگان بیشتری اضافه کنید.

### «آیا می‌توانم همزمان از چندین ارائه‌دهنده استفاده کنم؟**

**بله!** این تمام منظور RouteChi است. چندین ارائه‌دهنده متصل کنید و از `model: "auto"` استفاده کنید تا RouteChi برای هر درخواست بهترین را انتخاب کند.

### «اگر یک ارائه‌دهنده از کار بیفتد چه؟**

RouteChi به‌صورت خودکار ارائه‌دهندگان ناموفق را رد می‌کند و گزینه بعدی را امتحان می‌کند. نیازی به کاری ندارید.

### «چگونه یک ارائه‌دهنده را قطع کنم؟»

به Providers بروید → روی ارائه‌دهنده کلیک کنید → روی **Disconnect** کلیک کنید.

### «آیا می‌توانم از کلیدهای API موجودم استفاده کنم؟**

**بله!** اگر قبلاً کلیدهای API برای OpenAI، Anthropic، Google و غیره دارید، می‌توانید در RouteChi از آن‌ها استفاده کنید. کافیست هنگام اتصال ارائه‌دهنده آن‌ها را paste کنید.
